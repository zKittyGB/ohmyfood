# Project: Ohmyfood

## Context

This project was developed as part of my **OpenClassrooms training**.  
The goal was to practice using **Sass** and to create a responsive web application using HTML and CSS only.

---

## Technologies

- HTML5 & CSS3  
- Responsive design  
- CSS animations  
- Sass  
- FontAwesome  
- Semantic HTML structure  
- Google Fonts  

---

## Project Description

Ohmyfood is a web application showcasing a food-related interface with a focus on layout, styling, and animations.  
All styling is implemented using Sass, with responsive design for multiple screen sizes.
